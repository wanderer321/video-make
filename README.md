## 产品介绍

DramaForge（剧锻工坊）是一款面向短剧/漫剧创作者的 **本地桌面应用**，将 AI 能力整合进创作全流程——剧本创作、分镜规划、图像生成、配音合成、视频剪辑、成片导出——无需在多个工具之间来回切换。

**设计原则：**
- **本地优先**：所有数据（剧本、图像、视频、配置）全部存储在本地，不上传到任何第三方云服务
- **AI 接口自由**：可接入 Claude、OpenAI、可灵、Stability AI 等任意主流 AI 接口，也支持 ComfyUI/Ollama 等完全本地运行方案，零额外成本
- **全流程覆盖**：从一句话创意到可发布的 MP4 成片，流程全在一个界面内完成

## 核心功能

### 1. 项目管理

- 按项目组织漫剧创作，支持多集（分集）管理
- 项目类型：2D 漫剧、3D 漫剧、真人短剧
- 集数进度一览：每集的分镜数、已生成图像数、已生成视频数、当前状态

### 2. 剧本工坊

- **导入剧本**：支持拖拽上传 `.txt` / `.docx` / `.pdf` 文件，自动解析文本
- **AI 一键生剧本**：输入一句话题材描述（如"古风穿越复仇爽文"），AI 自动生成完整剧本框架，含分集梗概、角色设定、世界观、爽点规划
- **AI 剧本拆解**：上传现有剧本，AI 自动提炼角色列表（含外貌/性格）、世界观设定、分集梗概、核心矛盾
- **实时流式输出**：AI 生成过程实时逐字展示，无需等待全部完成
- **一键生成分镜画板**：拆解完成后，AI 自动将剧情转化为镜头序列，直接生成分镜画板

### 3. 资产库

- 管理三类创作资产：**角色**、**场景**、**道具**
- 支持 AI 自动生成参考图（可接入多种图像生成服务）
- 支持手动上传参考图
- 每个角色可配置专属 **TTS 配音声音**（音色、语速、提供商）
- 场景资产可自动注入分镜生成提示词，保证跨镜头视觉一致性
- 从剧本拆解结果一键导入角色到资产库

### 4. 分镜画板

- 可视化分镜卡片网格，按集数展示所有镜头
- 每个分镜卡片包含：分镜图预览、画面提示词、景别、机位、台词、时长、视频状态
- **单镜头模式切换**：每个镜头独立切换"分镜图优先"或"直接视频"模式，AI拆解时根据镜头内容自动推荐
- **批量生成分镜图**：对所有未生成图的分镜并行批量调用图像生成服务（3 路并发）
- **批量生成视频**：为所有分镜批量创建视频生成任务，并跳转至生成中心跟踪进度
- **批量配音**：按角色匹配声音配置，批量为所有有台词的分镜创建 TTS 配音任务
- **拖拽排序**：拖拽卡片调整镜头顺序，自动持久化
- **时长内联编辑**：点击时长数字直接修改，无需弹窗
- **提示词内联编辑**：双击画面描述区域直接修改提示词，保存后可重新生成
- **参考图关联**：自动匹配角色资产库中的参考图，支持手动修改

### 4.1 分镜规划页面架构

采用两级结构：
- **剧集列表页** (`/storyboard?project=...`)：显示项目所有剧集，卡片形式展示分镜数/状态
- **单集分镜页** (`/storyboard/episode/{episodeId}`)：左右两栏布局
  - 左栏：剧本内容展示（梗概/钩子/关键镜头）
  - 右栏：分镜操作区域（AI拆分镜 / 分镜表格）

**分镜表格列**：
| 编号 | 出镜主体 | 提示词 | 时长 | 模式 | 分镜图 | 操作 |

- 出镜主体：显示角色参考图缩略图（从资产库自动匹配）
- 提示词：左对齐显示，可点击展开编辑
- 模式：切换按钮（图片/视频），颜色区分
- 操作：生成按钮，调用图像/视频生成服务

### 5. 生成中心

统一管理所有 AI 生成任务的进度与结果：

- 实时任务状态推送（WebSocket，无需手动刷新）
- 任务类型：图像生成 / 视频生成 / 语音合成
- 状态追踪：排队中 → 生成中 → 已完成 / 失败
- 生成完成后可直接预览图片/视频/音频
- 一键下载生成结果
- 失败任务支持一键重试
- 成本估算（按 Provider 和时长/字数计算）
- 支持清除已完成任务
- 手动创建任务（可选图像/视频，选择 Provider，输入提示词）

### 6. 后期合成

基于分镜画板的时间轴编辑器：

- 可视化多轨道时间轴：视频轨、配音轨、字幕轨、背景音乐轨
- 时间轴缩放（10px/s ~ 120px/s），方便精细调整
- 分镜缩略图横向预览条，点击跳转播放位置
- **AI 配音面板**：在线合成台词配音，即时预览，支持选择音色
- **自动字幕**：调用 Whisper 语音识别自动转录配音为字幕文本，一键同步到分镜台词
- **字幕编辑**：点击片段直接编辑字幕文本
- **背景音乐**：上传本地音频作为 BGM，导出时自动混音（BGM 音量 15%）
- **一键导出 MP4**：调用本地 FFmpeg 拼接所有镜头，混合配音、BGM，烧录字幕，输出 9:16 竖屏格式（1080×1920）
- 导出完成后可在应用内直接下载成片

### 7. 设置中心

- 分类配置四大类 AI Provider：文本/剧本、图像生成、视频生成、配音/TTS
- 每个 Provider 独立配置 API Key、接口地址、默认模型等
- API Key 加密存储在本地，不明文保存
- 支持"测试连接"验证配置是否正确

## 快速启动

### 环境要求

| 组件 | 最低版本 | 说明 |
|------|----------|------|
| Python | 3.9+ | 运行后端 |
| Node.js | 18+ | 运行前端 |
| FFmpeg | 任意版本 | 视频导出必需，需加入 PATH |
| Rust + Cargo | 1.70+ | 仅打包桌面版时需要 |

### Step 1：安装后端依赖

```bash
cd backend
pip install -r requirements.txt
```

### Step 2：启动后端

```bash
cd backend
python main.py
# 后端监听 http://127.0.0.1:17321
```

### Step 3：安装前端依赖

```bash
cd desktop
npm install --registry https://registry.npmmirror.com
```

### Step 4：启动前端（浏览器预览模式）

```bash
cd desktop
npm run dev
# 访问 http://localhost:5173
```

### Step 5（可选）：打包为桌面应用

```bash
# 先安装 Rust
# 访问 https://rustup.rs 安装 rustup，然后：
rustup target add x86_64-pc-windows-msvc

# 打包
cd desktop
npm run tauri build
# 输出：desktop/src-tauri/target/release/bundle/
```

## AI 接口支持

### 文本/剧本生成（LLM）

| Provider | 说明 | 配置项 |
|----------|------|--------|
| Claude (Anthropic) | 剧本推理首选，中文效果极佳 | API Key、可选代理地址 |
| OpenAI / 兼容接口 | 支持 GPT-4o、DeepSeek、通义千问等 | API Key、Base URL、模型名 |
| DeepSeek | 高性价比国产大模型 | API Key、Base URL |
| 通义千问（阿里云） | 阿里云大模型 | API Key、模型名 |
| Ollama（本地） | 完全免费，零隐私风险 | 服务地址（默认 localhost:11434）、模型名 |

### 动态提示词增强系统

在剧本生成、剧集拆解、分镜拆解流程中，系统动态加载 `prompt-templates` 目录下风格特定的规则知识和例子，增强LLM提示词质量。

**支持的风格**（9种）：
- 历史权谋、古风玄幻、悬疑推理、灵异惊悚
- 科幻末世、逆袭爽剧、都市甜宠、青春校园

**增强内容注入**：
- 剧本生成：叙事规则（三幕结构、救猫咪节拍）、角色原型、风格例子
- 剧集拆解：分集节奏规则（"三集定生死"公式）、钩子设计
- 分镜拆解：Camera Contract规则、One-Move Rule、灯光设计、镜头策略判断规则

**镜头生成策略判断**（基于 `分镜生成方式详解.md`）：
- 连续运镜/表情变化 → 单套提示词（时间码分段）
- 景别变化/角度变化/场景转换 → 多套独立提示词
- gen_mode="video" → 直接生成视频

**详细文档**：见 `backend/services/README_PROMPT_ENHANCEMENT.md`

## 常见问题

**Q：后端启动后前端无法连接？**  
确认后端运行在 `http://127.0.0.1:17321`，浏览器访问 `http://localhost:17321/api/health` 应返回 `{"status":"ok"}`。

**Q：图像生成失败，提示"未配置"？**  
前往设置页，选择图像生成 Tab，配置至少一个 Provider 并保存。本地使用推荐先安装 ComfyUI，无需 API Key。

**Q：视频导出报错"FFmpeg 未安装"？**  
从 https://ffmpeg.org/download.html 下载 FFmpeg，将可执行文件目录加入系统 PATH，重启后端后重试。

**Q：Edge TTS 不需要配置就能用？**  
是的，Edge TTS 为免费服务，无需 API Key，开箱即用。在资产库为角色配置声音时选择"Edge TTS（免费）"即可。

**Q：如何完全本地运行（不花任何 API 费用）？**  
- LLM：安装 Ollama，运行 `ollama pull qwen2.5:7b`，在设置中选择 Ollama
- 图像生成：安装 ComfyUI（`localhost:8188`），在设置中选择 ComfyUI
- 配音：使用 Edge TTS（无需配置）或安装 CosyVoice
- 视频生成：目前暂无免费本地方案，可跳过视频生成步骤直接用图像导出

**Q：生成的 MP4 比例是什么？**  
默认输出 9:16 竖屏格式（1080×1920），适配抖音/快手/视频号短剧平台。
