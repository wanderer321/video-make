### 平台信息

| 维度 | 数据 |
|------|------|
| **官网** | https://www.stepfun.com |
| **定位** | AI视频/图片生成平台 |
| **特色** | StepClaw对话助手 + Step Audio Studio |
| **功能** | 网络搜索/知识问答/图片创作/语音ASR |

### 核心功能

```
StepClaw (对话助手):
- Web Search (网络搜索)
- Knowledge Base Q&A (知识问答)
- Image creation (图片创作)

- Deploy StepClaw (部署服务)
```

**用户反馈**: 阶跃视频提供预设主题和提示词模板

**预设主题推测** (基于官网):

**使用方式**:
1. 登录 stepfun.com

3. 使用预设主题模板

### 平台信息

| 维度 | 数据 |
|------|------|
| **官网** | https://krita.org |
| **定位** | 专业免费开源绘画软件 |

| **支持** | KDE + Qt框架 |
| **版本** | 5.3.0 / 6.0.0 (2026年3月) |
| **社区** | krita-artists.org |

### 核心特点

```
专业特性:
- 多种笔刷类型
- 图层、绘图助手、稳定器
- 动画工具
- 免费且开源
- 由艺术家制作
```

**来源**: Krita官方资源文档

**链接**: https://gumroad.com/l/CHhlx
**内容**:

```
构图模板包括:
- Rule of thirds (三分法)
- Golden ratio (黄金比例)
- Golden spiral (黄金螺旋)
- Golden triangle 1/2 (黄金三角)
- Harmonious triangle (和谐三角)
- Film safe area template (电影安全区域)
- Baroque diagonal (巴洛克对角线)
- Centre (中心构图)

特点:
- .zip压缩包格式
- Vector Library形式导入
- 导入后所有形状易于访问
```
