### 1. Skill保姆级教程 (5.6w播放)

**标题**: 保姆级教程！手把手教你写出好用又安全的专属Skill
**作者**: 阿真
**链接**: https://www.uisdc.com/skill

#### 📐 Skill核心结构

```yaml
skill-name/
├── SKILL.md          # 核心文件
├── scripts/          # 辅助脚本(可选)
├── references/       # 参考文档
└── assets/           # 素材资源
```

#### 🔑 Description写作公式

```yaml
# ❌ 太模糊
description: 帮忙处理文档

# ❌ 太宽泛
description: 帮助用户写文章

# ✅ 正确写法
description: |
  将故事文本转换为AI视频生成所需的分镜脚本。
  当用户说"做分镜"、"分镜脚本"、"故事转分镜"时触发。
```

**关键要点**:
- 功能描述 + 触发关键词
- 包含具体触发词和排除条件
- 避免模糊动词("帮助"、"支持"、"改善")

#### 📋 SKILL.md骨架

```markdown
## 核心原则
1. 用户不需要懂Skill格式
2. 每轮问完先小结确认
3. 允许跳过不确定的问题
4. 最终产出必须可直接使用

## 访谈阶段

### 第一轮：核心意图 (4个问题)
1. 这个Skill最终要产出什么？
2. 你平时会怎么说来触发它？
3. 哪些场景绝对不要触发？
4. 做到什么程度算完成？

### 第二轮：运行环境 (4个问题)
5. 运行在什么环境里？
6. 允许用哪些工具？
7. 需要读取哪些参考资料？
8. 需要写脚本吗？

### 第三轮：输出契约 (4个问题)
9. 最终输出必须长什么样？
10. 有哪些必须包含的内容？
11. 有哪些必须避免的内容？
12. 能给一个正例和一个反例吗？
```
