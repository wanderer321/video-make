### 2. 分镜制作专业流程 (Boords指南)

#### 四步分镜法

```markdown
## Step 1: 选择工具
- 线上工具：Boords、协作编辑、实时反馈
- 纸质模板：快速草图、现场参考、头脑风暴

## Step 2: 草绘帧格 (Scamping)
原则：保持粗糙、简单、不要过度细节

要点：
• 确定画幅比例 (16:9电影 / 9:16短视频)
• 保持粗略 - 不是艺术品
• 保持流动 - 检查叙事连贯性
• 注意连贯性 - 时间逻辑、状态延续

## Step 3: 编辑完善
视觉强化要素：
• 设定时间氛围 - 晨光/傍晚的情绪暗示
• 剪影检查 - 确保动作可识别
• 不抢角色风头 - 环景服务于叙事
• 景别多样性 - 长景/中景/特写交替
• 分层构图 - 前景/中景/背景层次

## Step 4: 添加摄像机运动
运动类型：
• Zoom (推拉) - 接近/远离主体
• Pan (摇镜) - 水平横向移动
• Tilt (俯仰) - 垂直上下移动
• Dolly (推车) - 轨道推进/后退
• Truck (横移) - 水平固定路径移动
• Pedestal (升降) - 垂直升降
```

#### 分镜术语表

```markdown
## 专业词汇

Aspect Ratio (画幅比例) - 宽高关系，定义画面形状
Frame (帧格) - 分镜单个画面框
Shot (镜头) - 连续动作序列
Scamps (草图) - 分镜初期粗糙草图
Script (脚本) - 视觉化前的文字起点
Shot List (镜头清单) - 每个镜头详细描述
Voiceover (旁白) - 非画面内角色朗读
Dialogue (对话) - 角色间交谈
Style Frame (风格帧) - 定调整体视觉的全彩图
Sound Effects (音效) - 环境音增强真实感
Animatic (动态分镜) - 配音轨的图像序列播放
```

## 🎥 AI动画参数库

### 视频生成参数详解

#### 画幅比例选择指南

| 比例 | 用途 | 特点 |
|------|------|------|
| **16:9** | YouTube、电视、企业视频 | 最常见，横向布局 |
| **2.39:1** | 电影长片、短片 | Cinemascope电影感 |
| **9:16** | TikTok、Reels、Shorts | 竖屏短视频 |
| **1:1** | Instagram、社交广告 | 方形紧凑 |
| **4:5** | Instagram竖版 | 平衡展示 |
| **21:9** | 超宽银幕 | 沉浸体验 |

#### 摄像机运动参数

| 运动类型 | 英文关键词 | 中文描述 | 效果 |
|----------|-----------|---------|------|
| **推镜** | `Zoom in` | 放大聚焦 | 强调主体 |
| **拉镜** | `Zoom out` | 缩小揭示 | 展示环境 |
| **摇镜** | `Pan left/right` | 水平转动 | 环览场景 |
| **俯仰** | `Tilt up/down` | 垂直转动 | 高度变化 |
| **推车** | `Dolly in/out` | 轨道推进 | 深度移动 |
| **横移** | `Truck left/right` | 水平滑动 | 平行跟随 |
| **升降** | `Pedestal up/down` | 垂直升降 | 高低视角 |
| **环绕** | `Orbit` | 围绕旋转 | 360度展示 |
| **跟拍** | `Tracking shot` | 跟随移动 | 动态叙事 |

#### 景别参数库

| 景别 | 英文关键词 | 画面占比 | 用途 |
|------|-----------|---------|------|
| **远景** | `Extreme Wide Shot (EWS)` | 主体<10% | 环境交代 |
| **全景** | `Wide Shot (WS)` | 主体~30% | 场景展示 |
| **中景** | `Medium Shot (MS)` | 主体~50% | 人物上半身 |
| **中近景** | `Medium Close-Up (MCU)` | 主体~70% | 面部+肩 |
| **特写** | `Close-Up (CU)` | 面部为主 | 情绪表达 |
| **大特写** | `Extreme Close-Up (ECU)` | 部位细节 | 强调重点 |
| **过肩** | `Over-the-Shoulder (OTS)` | 背后视角 | 对话场景 |
| **POV** | `Point of View` | 角色视角 | 代入感 |

#### 镜头角度参数

| 角度 | 英文关键词 | 效果 |
|------|-----------|------|
| **平视** | `Eye Level` | 自然平等 |
| **俯视** | `High Angle` | 弱化主体 |
| **仰视** | `Low Angle` | 强化权威 |
| **鸟瞰** | `Bird's Eye` | 全局俯瞰 |
| **斜角** | `Dutch Angle` | 不稳定紧张 |
| **虫视** | `Worm's Eye` | 极端仰视 |

#### 光线参数

| 光线类型 | 英文关键词 | 情绪效果 |
|----------|-----------|---------|
| **晨光** | `Morning light, dawn` | 希望、清新 |
| **午光** | `Midday, harsh sun` | 直接、强烈 |
| **黄昏** | `Golden hour, sunset` | 温暖、浪漫 |
| **夜光** | `Night, moonlight` | 神秘、悬疑 |
| **硬光** | `Hard lighting` | 戏剧性强 |
| **柔光** | `Soft lighting` | 温和自然 |
| **逆光** | `Backlit, silhouette` | 轮廓剪影 |
