## 📐 如何使用

### 1. 生成剧本

```
读取 → 风格-XXX/剧本-生成剧本&分集%%分片段-规则知识方法.md
        · 获取该风格的叙事模式、三幕结构映射、人物设定原则
        · 获取分镜设计规则（该风格专属的镜头语言）

参考 → 风格-XXX/剧本-例子.md
        · 参考真实剧集的故事结构、角色关系、叙事节奏

输出 → 按规则+例子生成该风格的新剧本
```

### 2. 生成分镜Prompt

```
判断 → 变化类型（运镜/景别/角度/场景/人物）
        - 运镜/表情/人物移动/时间连续 → 一套提示词
        - 景别/角度/场景转换/人物切换 → 多套提示词

读取 → 风格-XXX/动画/分镜-规则知识方法.md (or 真人视频)
        · Camera Contract、运镜公式、One-Move Rule
        · 灯光设计规则、物理语言清单

参考 → 风格-XXX/动画/分镜-例子.md (or 真人视频)
        · 对应方式的例子（一套或多套提示词）

输出 → 按规则组合输出分镜Prompt
```

### 3. 生成参考图Prompt

```
读取 → 风格-XXX/动画/参考图-规则知识方法.md (or 真人视频)
        · 色彩体系（主色调、辅色调、高光色）
        · 构图原则、光影规则

参考 → 风格-XXX/动画/参考图-例子.md (or 真人视频)
        · 角色、场景、道具的参考图Prompt实例

输出 → 按规则组合输出参考图Prompt
```

## ⚠️ 核心规则

- **规则知识方法文件**：纯创作理论/公式/规则/清单，不含完整例子
- **例子文件**：纯真实剧集的剧本/分镜/参考图例子，不含规则说明
- **生成流程**：规则文件做骨架指引 → 例子文件做素材参考 → 最终输出
- **所有例子**基于2024-2026年真实播出的爆款剧
