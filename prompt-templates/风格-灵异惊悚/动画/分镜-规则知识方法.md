## 📐 核心原则（Seedance 2.0 官方）

### Director's Formula
```
[Subject 主体] + [Action 动作] + [Scene 场景] + [Camera 运镜] + [Style 风格] + [Constraints 约束]
```
**灵异惊悚类型**: Drama/Narrative → 60-100字，以Scene引导

### One-Move Rule
**每个镜头只指定一个主要运镜**。堆叠多个运镜是生成失败的最常见原因。

### Camera Contract
```
Framing: [wide / medium / close-up / tight close-up / extreme close-up]
Move: [locked-off / dolly / pan / tilt / orbit / handheld / crane / tracking] (只选一个)
Speed: [slow / moderate / fast]
Angle: [eye level / low angle / high angle / dutch angle]
```

### 恐惧的灯光语言
| 恐惧类型 | 灯光方案 | 来源 |
|---------|---------|------|
| 未知恐惧 | `single bare bulb swinging overhead, deep shadows beyond 1m` | seedance-lighting |
| 密闭恐惧 | `candle practical, warm gold, deep shadow beyond 1m, thin fog catching the candlelight, visibility 2m` | seedance-lighting |
| 雨夜恐惧 | `firelight flicker, warm amber, unstable. rain visible in the firelight beams` | seedance-lighting |
| 月光恐惧 | `single moonlight beam through window as key, cool blue` | seedance-lighting |
| 实验室恐惧 | `cool fluorescent as key, screen blue fill` | seedance-lighting |

### 标准运镜短语
- `locked-off static camera, no movement`
- `slow dolly push from medium shot to tight close-up over 8 seconds`
- `slow dolly pull back revealing the full environment`
- `slow pan left revealing [new element]`
- `handheld tracking following the subject, subtle shake, not chaotic`
- `crane shot rising from ground level to overhead`

### 风格锚点
- `anamorphic, subtle film grain, muted palette`
- `vintage softness, warm highlights cold shadows`
- `low-key lighting, deep blacks, crushed shadows`

### VFX 特效合约（seedance-vfx）
```
Source: where does it originate?
Scale: size relative to subject
Behavior: drift / burst / swirl / dissipate / grow / shrink
Interaction: casts light / casts shadows / affects air / displaces objects
Duration: when does it start, peak, and fade?
```

### 音频设计（seedance-audio）
- 环境音：`The scene is silent except for the sound of wind.`
- 音效：`The sword makes a "shing" sound when drawn.`
- 对话口型同步：`Character A says: "We leave at dawn." Framing: medium close-up, locked-off camera.`
- 多人口型同步是Seedance 2.0官方承认的未解决问题，需要分开生成再合成

### 反 Slop 规则
- ❌ dramatic lighting → ✅ `candle practical, warm gold, deep shadow beyond 1m`
- ❌ tense atmosphere → ✅ `breath misting in cold air at each exhale`
- ❌ scary scene → ✅ `single bare bulb swinging overhead, deep shadows beyond 1m`
- ❌ epic horror → ✅ `firelight flicker, warm amber, unstable`
