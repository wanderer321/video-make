### 惊悚片的三幕结构映射
> **来源**: Syd Field, Screenplay (1979)

| 幕 | 惊悚片对应 | 唐诡示例 |
|------|-----------|---------|
| 第一幕（建制） | 诡异现象出现 | 古寺深夜惨叫 |
| 情节点一 | 主角决定调查 | 苏无名前往古寺 |
| 第二幕（对抗） | 更多诡异现象 | 鬼火、哭声、死亡 |
| 中点 | 发现人为线索 | 发现堂主私通 |
| 情节点二 | 最大危机 | 真凶发现被调查 |
| 第三幕（解决） | 真相大白 | 超自然现象被破解 |

## 📐 灵异惊悚类型创作指南

### 灵异惊悚对白设计原则
| 场景 | 对白特征 | 示例来源 |
|------|---------|---------|
| 发现诡异 | 紧张、压低声音 | 唐诡-古寺勘查 |
| 推理分析 | 冷静、逻辑清晰 | 唐诡-苏无名推理 |
| 惊吓时刻 | 简短、本能反应 | 灭门夜少年反应 |
| 真相揭露 | 平静、有条理 | 唐诡-真相大白 |
| 威胁对白 | 简短、有压迫感 | 藏海传-平津侯 |

### 诡案案件结构模板（每案6-8集）
```
第1-2集: 发现诡案
  - 诡异现象（死人/失踪/怪物）
  - 主角介入调查
  - 超自然表象建立

第3-4集: 调查取证
  - 收集线索
  - 排除嫌疑人
  - 更多诡异现象

第5-6集: 深入险境
  - 遇到危险
  - 发现关键线索
  - 真相逼近

第7-8集: 真相大白
  - 超自然现象被破解
  - 真凶落网
  - 人为阴谋揭露
```

## 📖 代表剧5:《唐诡之西行》(2025)

### 基本信息
- **主演**: 杨旭文（卢凌风）、杨志刚（苏无名）
- **平台**: 爱奇艺
- **类型**: 古装 · 悬疑 · 灵异探案
- **系列**: 唐朝诡事录系列第三季
- **备注**: 延续前作高质量，2025年悬疑探案爆款

### 完整故事概要
苏无名与卢凌风西行至沙州，卷入一系列诡异事件：沙漠中的鬼市、壁画里的美人复活、千年古尸还魂。每个案件都有超自然表象（鬼怪、复活的壁画美人），但最终用阴谋/人为解释。「表面灵异，实为人为」的核心模式延续至今。

**灵异看点**:
- 西部+灵异：沙漠、古墓、敦煌壁画的独特氛围
- 超自然表象下的阴谋：看似鬼怪实为人为
- 系列延续：角色成长和世界观累积

## 📖 代表剧6:《不思异·山魈》(2025)

### 基本信息
- **主演**: 王艺哲、张磊
- **平台**: B站
- **类型**: 悬疑 · 惊悚 · 都市传说
- **系列**: 不思异系列
- **集数**: 10集
- **备注**: 国产恐怖短剧系列口碑之作

### 完整故事概要
一档野外探险直播节目中，主播团队进入西南深山追踪传说中的「山魈」。最初以为是节目效果，但队员接连失踪，直播信号中出现诡异影像。摄像机记录的片段越来越离奇——山魈不杀人，但会「替换」人。当你发现身边的队友已经不是原来那个人时，恐惧才真正开始。

**灵异看点**:
- 伪纪录片：直播形式增强真实感
- 心理学恐怖：不是突然吓人，而是细思极恐
- 本土志怪：中国山魈传说 vs 西方恐怖

## 📖 代表剧7:《夜不语》(2025)

### 基本信息
- **主演**: 陈博豪、屠芷莹
- **平台**: 爱奇艺
- **类型**: 悬疑 · 惊悚 · 都市
- **原著**: 夜不语小说系列
- **集数**: 12集

### 完整故事概要
大学讲师夜不语被卷入一系列违背科学常理的离奇事件：一个废弃医院里，有人死后还持续工作；一部旧手机总能收到来自「已故者」的短信；一座公寓里居民开始集体遗忘自己的身份。每集一个独立案件，逐步串联成更大阴谋。

**灵异看点**:
- 科学视角：理性主角用科学方法调查灵异事件
- 都市空间恐怖：废弃医院、旧公寓、深夜地铁
- 细思极恐：没有鬼但比有鬼更可怕

## 📖 代表剧8:《她死在了昨晚》(2026)

### 基本信息
- **主演**: 张雅钦、黄俊捷
- **平台**: 腾讯视频
- **类型**: 悬疑 · 惊悚 · 都市爱情
- **集数**: 18集
- **备注**: 2026年开年头号惊悚短剧

### 完整故事概要
女主角每晚12点准时在公寓电梯里「死亡」——同样的时间、同样的地点、同样的方式，然后第二天醒来一切如常。男主角是唯一相信她的人，两人必须在每晚的「12点循环」中找出真相。最终发现这并不是灵异事件，而是一场被精心设计的心理操控实验。

**灵异看点**:
- 时间循环+惊悚：重复经历死亡的恐怖
- 心理操控：灵异背后的科学解释
- 爱情线：在绝望中互相拯救

## 📐 灵异惊悚类型创作补充指南

### 恐怖类型金字塔（从低成本到高成本）
| 层级 | 恐怖类型 | 成本 | 代表作品 |
|------|---------|------|---------|
| 心理恐怖 | 环境、氛围、音乐 | 低 | 不思异系列 |
| 都市传说 | 日常空间中的异常 | 低 | 夜不语 |
| 伪纪录片 | 手持/录播/监控视角 | 低 | 不思异·山魈 |
| 悬疑探究 | 表面灵异实为阴谋 | 中 | 唐朝诡事录 |
| 视觉恐怖 | 怪物、鬼怪实体化 | 高 | 鬼吹灯系列 |

### 灵异惊悚的「恐怖递进」公式
```
第一层（暗示）: 异常但不恐怖的声音/画面
  - 灯光闪烁、手机信号异常、温度骤降
第二层（确认）: 确认异常存在
  - 镜子里多了一个人、听见不存在的声音
第三层（逼近）: 威胁感逼近主角
  - 门自己开了、衣柜在晃动
第四层（高潮）: 直面恐惧
  - 亲眼见到鬼/怪物/真相
第五层（余韵）: 恐惧的延续
  - 结尾暗示异常仍未消除
```

### 灵异对白设计原则
| 场景 | 对白特征 | 示例 |
|------|---------|------|
| 发现异常 | 试探性、轻声、不确定 | 「你…听到了吗？」 |
| 恐慌升级 | 短促、重复、呼吸急促 | 「它又来了」 |
| 调查分析 | 理性分析vs非理性遭遇 | 「一定有科学解释」 |
| 直面恐惧 | 决定性的简短宣言 | 「我们进去」 |
| 事后回忆 | 颤抖、停顿、欲言又止 | 「那个东西…不是人」 |

### 「表面灵异，实为人为」结构的戏剧优势
| 优势 | 说明 | 示例 |
|------|------|------|
| 前期吸引 | 灵异表象吸引猎奇观众 | 「鬼怪」增加话题度 |
| 后期反转 | 人为揭秘带来智力快感 | 「原来是这样！」 |
| 低成本 | 不依赖鬼怪特效 | 暗示比展示更可怕 |
| 社会评论 | 借灵异外壳讽刺现实 | 心理操控实验的隐喻 |
| 系列化潜力 | 每次都是不同案件 | 唐朝诡事录-单元剧模式 |


### 灵异惊悚分镜设计规则

#### 恐怖递进的镜头公式
| 阶段 | 镜头方案 | 持续时间 | 示例 |
|------|---------|---------|------|
| 暗示阶段 | 环境空镜（走廊尽头/半开的门/水面倒影），镜头缓慢平稳 | 3-5s | 走廊尽头一扇门慢慢关上 |
| 确认阶段 | 人物反应镜头（特写面部）→ 人物视线方向 → POV看到异常 | 2-3s | 角色看到镜中多了一个影子 |
| 逼近阶段 | 跟拍角色后退 → 前景物体被推动/晃动的过肩镜头 | 4-6s | 衣柜门慢慢打开，角色后退 |
| 惊吓阶段 | 快速推近（zoom in）+ 音效爆点 + 随即跳切 | 1s | 鬼脸突然出现在镜头前 |
| 余韵阶段 | 缓慢拉远大全景，角色蜷缩 | 5s+ | 从惊恐的角色拉至房间全景 |

#### 低成本恐怖分镜技法
| 技法 | 做法 | 效果 |
|------|------|------|
| 声音先于画面 | 先听到异常声音，角色转头前切 | 想象比看到更恐怖 |
| 信息遮挡 | 用门框/窗框/书架遮挡视线 | 未知产生恐惧 |
| 镜面反射 | 不直接拍鬼，拍镜中的异常 | 心理暗示 |
| 手电筒光照 | 唯一光源是角色手电，扫过黑暗区域 | 未知恐惧最大化 |
| 监控视角 | 固定广角镜头，客观冷峻 | 不可靠的客观性 |
| 反套路静默 | 观众预期惊吓时故意不给，延迟释放 | 累积紧张>瞬间释放 |

#### 夜间恐怖场景的布光方案
| 场景 | 布光方案 | 难度 |
|------|---------|------|
| 废弃建筑搜索 | 手持LED灯作为唯一光源+演员自打光 | 低 |
| 卧室深夜 | 窗外月光（LED蓝光板）+ 床底黑暗阴影 | 低 |
| 走廊追逐 | 顶部频闪灯+手持跟拍 | 中 |
| 地下室/地窖 | 手电筒实拍+后期提亮阴影 | 低 |
| 雨中墓地 | 雨夜路灯（唯一的暖光）+ 大面积冷色 | 中 |
