## 📐 核心原则（Seedance 2.0 官方）

### Director's Formula
```
[Subject 主体] + [Action 动作] + [Scene 场景] + [Camera 运镜] + [Style 风格] + [Constraints 约束]
```
**科幻类型**: Drama/Narrative → 60-100字 | **风景类型**: Landscape → 30-60字

### One-Move Rule
**每个镜头只指定一个主要运镜**。堆叠多个运镜是生成失败的最常见原因。

### Camera Contract
```
Framing: [wide / medium / close-up / tight close-up / extreme close-up]
Move: [locked-off / dolly / pan / tilt / orbit / handheld / crane / tracking] (只选一个)
Speed: [slow / moderate / fast]
Angle: [eye level / low angle / high angle / dutch angle]
```

### 类型运镜预设
| 类型 | 推荐运镜 | 避免 |
|------|---------|------|
| 风景/旅行 | slow aerial pullback, slow pan, static wide | handheld, fast moves |
| 戏剧/叙事 | slow push-in, dolly pull-out, tracking, static | fast orbit, snap zoom |
| 产品展示 | orbit, slow push-in, static | handheld, whip pan |
| 生活/社交 | handheld, static, slow pan | crane, dolly zoom |

### 标准运镜短语
- `locked-off static camera, no movement`
- `slow dolly push from medium shot to tight close-up over 8 seconds`
- `slow dolly pull back revealing the full environment`
- `slow pan left revealing [new element]`
- `slow aerial pullback revealing the vast scale`
- `handheld tracking following the subject, subtle shake, not chaotic`

### 灯光设计
- ❌ dramatic lighting → ✅ `blue plasma flame as key light, cool blue`
- ✅ `cold fluorescent as key, screen blue fill`
- ✅ `overcast diffused daylight, soft wrap shadows`
- ✅ `single bare bulb swinging overhead, deep shadows beyond 1m`

### 大气效果
- `thin fog catching the plasma light, visibility 500m`
- `dust motes suspended in a sunbeam from camera-left`
- `breath misting in cold air at each exhale`
- `heat shimmer rising from asphalt, subtle`
- `smoke drifting from burning buildings, backlit by fire`

### 风格锚点
- `anamorphic, subtle film grain, muted palette`
- `vintage softness, warm highlights cold shadows`
- `low-key lighting, deep blacks, crushed shadows`
- `neon-saturated palette, cyberpunk aesthetic`

### VFX 特效合约（seedance-vfx）
```
Source: where does it originate?
Scale: size relative to subject
Behavior: drift / burst / swirl / dissipate / grow / shrink
Interaction: casts light / casts shadows / affects air / displaces objects
Duration: when does it start, peak, and fade?
```

### 反 Slop 规则
- ❌ epic sci-fi scene → ✅ `blue plasma flame as key light, cool blue, thin fog catching the plasma light`
- ❌ dramatic space scene → ✅ `cold fluorescent as key, screen blue fill, dust motes suspended in the light`
- ❌ tense atmosphere → ✅ `breath misting in cold lab air at each exhale`
- ❌ cinematic → 删除（默认就是cinematic）
