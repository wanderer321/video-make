# 分镜生成方式 - 知识与规则详解

## 一套提示词 vs 多套提示词的区别

---

## 📚 知识要点

### 方式一：一套提示词生成多张分镜图

**适用场景**：
- **连续镜头**：镜头之间有连续性关系
- **相同场景**：在同一场景内的多个镜头
- **相同人物**：人物位置/表情有连续变化
- **相似构图**：构图框架相同，只有细节变化
- **运镜变化**：运镜变化但其他参数相同

**特点**：
- 提示词包含时间码或分段标记
- AI可以理解连续性关系
- 生成的分镜图有连贯性
- 适合：运镜变化、人物移动、表情变化

**提示词格式**：
```
[时间码] 场景描述 | [时间码] 场景描述 | [时间码] 场景描述

示例：
[0-2秒] 中景，主角站立咖啡馆 | [2-4秒] 中景，主角走向柜台 | [4-6秒] 中景，主角到达柜台
```

---

### 方式二：多套不同提示词生成多张分镜图

**适用场景**：
- **不同景别**：远景→中景→特写的变化
- **不同角度**：平视→俯视→仰视的变化
- **不同场景**：场景转换的情况
- **不同人物**：人物切换的情况
- **完全不同构图**：构图框架完全不同

**特点**：
- 每套提示词独立完整
- AI需要分别生成每张图
- 生成的分镜图独立性强
- 适合：景别变化、角度变化、场景转换、人物切换

**提示词格式**：
```
提示词1: 远景，咖啡馆全景，无人物
提示词2: 中景，主角在柜台前，微笑
提示词3: 特写，主角面部表情，开心微笑
```

---

## ⚠️ 规则总结（高压线）

### 必须使用一套提示词的情况

- ✅ **运镜变化**：推镜/拉镜/摇镜等运镜变化
- ✅ **人物移动**：人物在同一场景内移动
- ✅ **表情变化**：表情有连续变化趋势
- ✅ **时间连续**：镜头时间有连续性

### 必须使用多套提示词的情况

- ✅ **景别变化**：远景→中景→特写等景别变化
- ✅ **角度变化**：平视→俯视→仰视等角度变化
- ✅ **场景转换**：场景转换到不同地点
- ✅ **人物切换**：人物切换到不同人物
- ✅ **构图完全不同**：构图框架完全不同

### 判断规则

| 变化类型 | 一套提示词 | 多套提示词 |
|----------|-----------|-----------|
| 运镜变化 | ✅ 适用 | ❌ 不适用 |
| 景别变化 | ❌ 不适用 | ✅ 适用 |
| 角度变化 | ❌ 不适用 | ✅ 适用 |
| 人物移动（同场景） | ✅ 适用 | ❌ 不适用 |
| 场景转换 | ❌ 不适用 | ✅ 适用 |
| 表情变化（连续） | ✅ 适用 | ❌ 不适用 |
| 人物切换 | ❌ 不适用 | ✅ 适用 |

## 📚 实战例子对比

### 例子1：运镜变化（一套提示词）

**场景**：主角走向咖啡馆柜台

**错误做法（多套提示词）**：
```
提示词1: 中景，主角站立咖啡馆入口，看向柜台
提示词2: 中景，主角走向柜台，步伐坚定
提示词3: 中景，主角到达柜台，微笑
```

**正确做法（一套提示词）**：
```
一套提示词: [0-2秒] 中景固定镜头，主角站立咖啡馆入口，看向柜台 | [2-5秒] 中景跟拍镜头，主角走向柜台，步伐坚定 | [5-8秒] 中景固定镜头，主角到达柜台，微笑

AI生成结果：3张连续分镜图，人物位置连续变化，保持连贯性
```

---

### 例子2：景别变化（多套提示词）

**场景**：从远景到特写的景别变化

**错误做法（一套提示词）**：
```
一套提示词: [0-2秒] 远景，咖啡馆全景 | [2-4秒] 中景，主角在柜台 | [4-6秒] 特写，主角面部
```

**正确做法（多套提示词）**：
```
提示词1: 远景，咖啡馆全景，环境展示，无人物
提示词2: 中景，主角在柜台前，微笑表情，与服务员对话
提示词3: 特写，主角面部表情，开心微笑，眼睛明亮

AI生成结果：3张独立分镜图，景别完全不同，构图框架完全不同
```

---

### 例子3：场景转换（多套提示词）

**场景**：从咖啡馆转换到街道

**错误做法（一套提示词）**：
```
一套提示词: [0-2秒] 中景，咖啡馆内部，主角喝咖啡 | [2-4秒] 远景，街道外部，主角走出咖啡馆
```

**正确做法（多套提示词）**：
```
提示词1: 中景，咖啡馆内部，主角喝咖啡，温馨氛围，明亮光线
提示词2: 远景，街道外部，主角走出咖啡馆，黄昏光线，城市环境

AI生成结果：2张独立分镜图，场景完全不同，环境完全不同
```

---

### 例子4：人物表情连续变化（一套提示词）

**场景**：主角表情从惊讶到开心

**正确做法（一套提示词）**：
```
一套提示词: [0-2秒] 特写，主角面部，惊讶表情，眼睛睁大 | [2-4秒] 特写，主角面部，微笑开始，嘴角微微上扬 | [4-6秒] 特写，主角面部，开心微笑，眼睛明亮

AI生成结果：3张连续分镜图，表情有连续变化趋势，保持连贯性
```

### 例子5：角度变化（多套提示词）

**场景**：从平视到俯视的角度变化

**正确做法（多套提示词）**：
```
提示词1: 中景，平视角度，主角站立，正面视角
提示词2: 中景，俯视角度，主角站立，上方视角，人物显得渺小

```

## ✅ 使用建议

### 快速判断流程图

```
是否需要分镜图？
↓
是否有连续镜头？
↓
是否有运镜变化？ → YES → 一套提示词（时间码）
↓
是否有景别变化？ → YES → 多套提示词（独立）
↓
是否有角度变化？ → YES → 多套提示词（独立）
↓
是否有场景转换？ → YES → 多套提示词（独立）
↓
是否有人物切换？ → YES → 多套提示词（独立）
↓
是否是人物移动/表情变化？ → YES → 一套提示词（时间码）
↓
其他情况 → 多套提示词（安全）
```

---

> **核心总结**：
> - 一套提示词：连续性关系（运镜/移动/表情）
> - 多套提示词：独立性关系（景别/角度/场景/人物）
