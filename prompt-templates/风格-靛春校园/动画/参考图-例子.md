## ✅ 靛春校园动画参考图设计总结

| 核心要素 | 说明 |
|---------|------|
| 色彩主调 | 靛蓝 + 暖橙/暖黄 |
| 情感光谱 | 从克制到温暖 |
| 场景偏好 | 办公室、咖啡馆、便利店、天台、地铁、公寓 |
| 构图重点 | 双人镜头、环境渲染、情感氛围 |
| 光影风格 | 夜景暖光、黄昏逆光、室内柔光 |
| 情绪递进 | 孤独→相遇→心动→考验→相守→释然 |

### 动画 vs 真人参考图差异
| 维度 | 动画参考图 | 真人参考图 |
|------|----------|----------|
| 色彩表达 | 更夸张的靛蓝/暖橙对比，饱和度可更高 | 更写实的肤色和光影，低饱和度 |
| 情感渲染 | 用画面氛围（光影/色彩/构图）直接表达情绪 | 用演员的微表情和自然光影暗示情绪 |
| 场景设计 | 完美理想化的都市空间 | 真实存在的拍摄场地 |
| 成本优势 | 省钱：人设画好即可换场景 | 费钱：每个场景都需要实拍或搭景 |
| 表现形式 | 2D/3D皆可，色彩信息传达能力强 | 摄影机位、灯光、化妆、服装综合决定 |
