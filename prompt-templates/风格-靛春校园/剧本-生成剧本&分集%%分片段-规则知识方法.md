## 📐 靛春校园风格创作指南

### 风格定义
靛春校园 = 成熟都市爱情剧，不是校园青春剧。核心区别：

| 维度 | 青春校园 | 靛春校园 |
|------|---------|---------|
| 主角年龄 | 16-22岁 | 25-35岁 |
| 场景 | 教室、宿舍、操场 | 办公室、公寓、社交场所 |
| 爱情模式 | 初恋、暗恋、表白 | 重逢、合作关系、职场爱恋 |
| 冲突来源 | 学业、友情、家庭 | 职场竞争、现实压力、心理创伤 |
| 成长主题 | 从孩子到大人 | 从大人到更好的自己 |
| 性张力 | 含蓄、懵懂 | 克制、张力、成年人的暧昧 |

### 三幕结构映射

| 幕 | 靛春校园对应 | 你比星光美丽示例 |
|------|------------|----------------|
| 第一幕（建制） | 重逢/意外相遇 → 被迫合作 | 纪星辞职创业，重逢韩廷 |
| 情节点一 | 合作关系确立 | 韩廷投资纪星公司 |
| 第二幕（对抗） | 职场合作 + 感情萌芽 | 一起创业、攻克难关 |
| 中点 | 感情明朗化 | 韩廷表白/纪星心动 |
| 情节点二 | 现实考验 | 公司危机/竞争对手/价值观冲突 |
| 第三幕（解决） | 守住初心 → 最终在一起 | 事业成功，感情修成正果 |

### 对白设计原则
| 场景 | 对白特征 | 示例来源 |
|------|---------|---------|
| 职场 | 专业、理性 | 你比星光美丽-创业会议 |
| 暧昧期 | 试探、暗示 | 三分野-重逢对话 |
| 冲突 | 克制但锋利 | 在暴雪时分-争吵 |
| 表露心意 | 不动声色地深情 | 照亮你-十年暗语 |
| 亲密 | 自然、有成熟感 | 请和这样的我恋爱吧 |

### 视觉风格关键词
- 色彩：靛蓝/深蓝为主色调，配暖色点缀（黄昏、灯光）
- 构图：大量中景和双人镜头，展现成年人之间的距离感
- 光影：柔和暖光（室内）、自然蓝调（室外傍晚）
- 场景：高端公寓、精致咖啡馆、CBD办公室、深夜便利店


### 靛春校园分镜设计规则

#### 成年人爱情的距离感镜头
| 关系阶段 | 镜头距离 | 构图特征 | 代表剧参考 |
|---------|---------|---------|----------|
| 初见/重逢 | 双人镜头中间有间隙(物理距离) | 物体隔在两人之间(桌子/柱子) | 你比星光美丽-重逢 |
| 试探阶段 | 间隙缩小但仍有 | 过肩对话，但互相不看对方 | 三分野-尴尬期 |
| 暧昧阶段 | 间隙消失，开始同框 | 身体微微倾向对方的方向 | 在暴雪时分-心动期 |
| 确认关系 | 亲密距离，身体的接触 | 对称构图，两人一起入镜 | 照亮你-在一起后 |
| 成熟相守 | 自然的家庭距离 | 一人做自己的事，另一个人也在的日常 | 请和我这样的恋爱吧 |

#### 都市情感剧场景模板
| 场景 | 分镜方案 | 情绪目的 |
|------|---------|---------|
| 深夜便利店 | 蓝色夜景外景(2s) → 便利店暖光内景(2s) → 两人各自买夜宵在收银台相遇(6s) | 城市孤独感中的暖意 |
| 加班后回家路 | 城市夜景延时叠化(3s) → 一人走在空荡街头的跟拍(8s) → 路灯下拉长的影子(2s) | 独居都市人的日常 |
| 厨房一起做饭 | 两人在灶台前自然互动的全景(5s) → 递调料时手碰到一起(3s) | 同居/关系升温的标志 |
| 阳台深夜谈心 | 背对城市夜景的双人剪影(4s) → 各说心事的侧脸(各3s) → 沉默享受夜景(5s) | 深层次的交流 |
| 周末逛超市 | 推车过道的广角跟拍(5s) → 一起挑选物品的中景(4s) → 往车里扔对方爱吃的零食(3s) | 最朴素的幸福 |

#### 成年人情感戏的克制表达
| 情感 | 分镜表达 | 避免 |
|------|---------|------|
| 吃醋 | 看到对方和别人亲近时，手中动作停住(2s) → 若无其事继续(3s) | 夸张的瞪眼/质问 |
| 感动 | 对方转身后，自己才露出微笑 | 当面流泪 |
| 思念 | 深夜摸出手机打开对话框 → 打了又删 → 放下手机看天花板 | 旁白说「我好想你」 |
| 决定分手 | 说分开的人语气平淡→切到桌下握紧到发白的手 | 撕心裂肺的哭戏 |
| 和好 | 不说「对不起」— 买对方爱吃的/做对方喜欢的事 | 长篇大论的道歉 |
