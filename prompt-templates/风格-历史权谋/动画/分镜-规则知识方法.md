## 📐 核心原则：Seedance 2.0 官方提示词公式

> **来源**: [seedance-prompt SKILL.md](https://github.com/Emily2040/seedance-2.0/blob/main/skills/seedance-prompt/SKILL.md)

### Director's Formula（导演公式）
```
[Subject 主体] + [Action 动作] + [Scene 场景] + [Camera 运镜] + [Style 风格] + [Constraints 约束]
```

### 各类型长度目标
| 类型 | 引导部分 | 目标字数 | 关键段落 |
|------|---------|---------|---------|
| 产品展示 | Subject | 30-50字 | Subject, Camera, Style |
| 生活/社交 | Action | 40-60字 | Action, Scene, Style |
| 戏剧/叙事 | Scene | 60-100字 | Scene, Action, Camera |
| 音乐视频 | Style | 50-80字 | Style, Action, Camera |
| 风景/旅行 | Scene | 30-60字 | Scene, Camera, Style |
| 商业品牌 | Style | 40-70字 | Style, Scene, Action |
| 动漫/艺术 | Style | 50-90字 | Style, Action, Scene |

### I2V vs T2V 模式
- **I2V 模式**（有参考图）：提示词只需描述**动作和运镜**，不要重复描述静态视觉元素
- **T2V 模式**（无参考图）：提示词必须从头描述所有视觉元素

### 物理语言（不用模糊词）
- ❌ "dynamic camera movement" → ✅ "slow dolly push from medium shot to tight close-up over 8 seconds"
- ❌ "beautiful lighting" → ✅ "single overhead practical as hard key, 5600K, deep shadow fill"
- ❌ "epic battle" → ✅ "dust erupts violently, sweat flies off in slow motion, fat renders slowly"

## 🎥 核心原则：One-Move Rule（单运镜规则）

> **来源**: [seedance-camera SKILL.md](https://github.com/Emily2040/seedance-2.0/blob/main/skills/seedance-camera/SKILL.md)

**每个镜头只指定一个主要运镜**。不要堆叠多个运镜（如 dolly push + pan left + tilt up），这是导致抖动、相机旋转和生成失败的最常见原因。

### Camera Contract（运镜合约）
每个镜头应包含：
```
Framing: [wide / medium / close-up / tight close-up / extreme close-up]
Move: [locked-off / dolly / pan / tilt / orbit / handheld / crane / tracking] (只选一个)
Speed: [slow / moderate / fast / "over 8 seconds"]
Angle: [eye level / low angle / high angle / dutch angle]
```

### 类型运镜预设
| 类型 | 推荐运镜 | 避免 |
|------|---------|------|
| 产品展示 | orbit, slow push-in, static | handheld, whip pan |
| 生活社交 | handheld, static, slow pan | crane, dolly zoom |
| 戏剧叙事 | slow push-in, dolly pull-out, tracking, static | fast orbit, snap zoom |
| 音乐视频 | whip pan, snap zoom, fast tracking, orbit | slow pan |
| 风景旅行 | slow aerial pullback, slow pan, static wide | handheld, fast moves |
| 商业品牌 | tracking, crane up, slow push-in | shaky handheld |
| 动漫艺术 | dynamic low-angle, fast push-in, whip pan | subtle, slow moves |

### 标准运镜短语（确保清晰）
- `locked-off static camera, no movement`
- `slow dolly push from medium shot to tight close-up over 8 seconds`
- `slow dolly pull back revealing the full environment`
- `slow pan left revealing [new element]`
- `slow tilt up from [foreground] to [sky]`
- `slow orbit left around the subject, constant distance`
- `handheld tracking following the subject, subtle shake, not chaotic`
- `crane shot rising from ground level to overhead`

## 💡 核心原则：灯光设计

> **来源**: [seedance-lighting SKILL.md](https://github.com/Emily2040/seedance-2.0/blob/main/skills/seedance-lighting/SKILL.md)

### 命名光源（必须指定）
- ❌ dramatic lighting → ✅ `single overhead practical as hard key, 5600K, deep shadow fill`
- ✅ `window backlight casting long shadows toward camera`
- ✅ `neon sign as key light, pink and blue, no fill`
- ✅ `firelight flicker, warm amber, unstable`
- ✅ `overcast diffused daylight, soft wrap shadows`
- ✅ `single bare bulb swinging overhead`
- ✅ `candle practical, warm gold, deep shadow beyond 1m`
- ✅ `streetlamp sodium key, cool fill from ground reflection`

### 核心灯光参数
| 参数 | 选项 | 说明 |
|------|------|------|
| Key direction（主光方向） | camera-left / camera-right / above / below / behind (rim) | 主光源方向 |
| Contrast（对比度） | low-key (deep shadows) / high-key (bright) | 光影对比 |
| Color temp（色温） | warm amber / cool blue / neutral white | 色温（Kelvin可选） |
| Shadow edge（阴影边缘） | hard-edged / soft wrap / no shadows | 阴影硬度 |

### 大气效果（必须用物理描述）
- `thin fog catching the backlight, visibility 5m`
- `dust motes suspended in a sunbeam from camera-left`
- `light rain visible against the key light, not heavy`
- `breath misting in cold air at each exhale`
- `heat shimmer rising from asphalt, subtle`
- `smoke drifting from off-screen right, backlit`

### 灯光过渡（用时间戳）
```
0-3s: warm golden overhead light.
3-5s: lights flicker and die.
5-end: only red emergency lamp remains; cool rain spill from open door as rim.
```

### 中英灯光术语对照
| 中文 | 英文 | 用法 |
|------|------|------|
| 主光 | Key light | 主要光源 |
| 辅光 | Fill light | 柔化阴影 |
| 轮廓光 | Rim light | 边缘分离 |
| 逆光 | Backlight / Contre-jour | 剪影或边缘光 |
| 低调布光 | Low-key | 暗调高对比，戏剧 |
| 高调布光 | High-key | 明亮低对比，商业 |
| 实景灯光 | Practical light | 画面中可见光源 |
| 丁达尔效应 | Tyndall rays / God rays | 光束穿过粒子 |

## 🎨 核心原则：视觉风格

> **来源**: [seedance-style SKILL.md](https://github.com/Emily2040/seedance-2.0/blob/main/skills/seedance-style/SKILL.md)

### 风格预算：1个主要锚点
推荐只用1个主要风格锚点，最多2个。超过3个会导致注意力稀释。

### 风格锚点（用电影语言，不用流行词）
| 维度 | 选项 |
|------|------|
| 镜头感 | anamorphic / vintage softness / spherical / fisheye |
| 纹理 | subtle film grain / digital clean / noise as character |
| 调色 | muted / desaturated / warm highlights cold shadows / neon-saturated |
| 对比度 | low-key / high-key / deep blacks / crushed shadows |

### 时期美学（用材料和灯光描述，不用年代标签）
| 时期 | 材料和灯光 |
|------|-----------|
| 1920s | oil lamp practicals, soot-stained plaster, handwoven wool, iron hardware |
| 1970s | film stock warm yellows, faded contrast, wide collars, grain heavy |
| 1990s | VHS scan lines, oversaturated color, handheld shake |
| 古代封建 | rough-hewn timber, candle light, raw silk, bronze fittings |

### CGI材质合约（避免塑料感）
每个材质指定2-4个属性：
```
Base: metal / painted metal / glass / ceramic / rubber / fabric / wood / stone
Roughness: matte / satin / glossy / mirror
Imperfection: micro-scratches / dust / fingerprints / wear marks / patina
Edge: slightly beveled / razor sharp / rounded / chipped
```

**示例**:
- `brushed aluminum, satin roughness, fine micro-scratches, subtle edge wear`
- `aged leather, matte surface, visible grain, creased at flex points`

### 材质运动物理
| 材质 | 运动特征 |
|------|---------|
| 重物 | 慢加速，慢减速 |
| 布料 | 滞后于运动，回弹过冲 |
| 玻璃 | 反光随运镜变化 |
| 液体 | 惯性晃动，缓慢稳定 |

### 风格迁移（最可靠方法）
- `Match the visual style, color grading, and film texture of @Video1`
- `Apply @Image1's artistic style and color palette to the scene`

模型会提取：调色、对比度、胶片颗粒、灯光氛围、构图风格、剪辑节奏。

## 🏃 核心原则：动作控制

> **来源**: [seedance-motion SKILL.md](https://github.com/Emily2040/seedance-2.0/blob/main/skills/seedance-motion/SKILL.md)

### One Action Per Shot（每镜头一个动作）
不要在一个镜头中链接多个不同动作。

### 打斗场景：意图+后果描述法
```
Characters: A references @Image1; B references @Image2.

Shot 1: A throws a right hook at B's jaw.
Shot 2: B ducks under the punch and sweeps A's legs.
Shot 3: A jumps, landing a spinning back kick to B's shoulder.
Shot 4: B staggers backward two steps, recovering his balance.

Camera: Medium shot, tracking the action. Slight handheld shake on impacts.
Physics: Dust puffs up from the ground on the leg sweep. A wet impact sound accompanies each hit.
```

### 参考视频法（最可靠）
```
// Recommended Prompt Structure
Characters: A references @Image1; B references @Image2.
Choreography: The archer fires two arrows; the mage deflects them with a violet energy shield, then closes distance and blasts the archer into a tree with a shockwave.
Reference: Reference the fight actions, character movements, and camera work from @Video1.
Style: Match the gritty, handheld style of @Video1.
```

### 节拍密度
模型通常每5秒能处理1-2个不同节拍。高密度需要微编排格式。
