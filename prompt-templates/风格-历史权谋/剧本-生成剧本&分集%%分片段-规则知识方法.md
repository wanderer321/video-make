## 📐 编剧理论参考：三幕结构在权谋剧中的应用

> **来源**: Syd Field《Screenplay: The Foundations of Screenwriting》(1979)
> 三幕结构是电影和电视剧创作中最广泛使用的叙事框架。

### 三幕结构定义
根据Syd Field的理论，一个标准剧本分为三幕：

| 幕 | 功能 | 占全剧比例 |
|------|------|-----------|
| 第一幕（建制） | 介绍角色、世界观、冲突起因 | 前25% |
| 第二幕（对抗） | 主角面对障碍，冲突升级 | 中间50% |
| 第三幕（解决） | 高潮对决，冲突解决 | 后25% |

### 关键情节点
- **情节点一**（第一幕结尾）：事件迫使主角采取行动，故事真正开始
- **中点**（第二幕中间）：重大转折，主角从被动变为主动
- **情节点二**（第二幕结尾）：最大危机，主角跌入谷底
- **高潮**（第三幕结尾）：最终对决，问题解决

### 救猫咪（Save the Cat）15个节拍
> **来源**: Blake Snyder《Save the Cat!》(2005)

Blake Snyder将剧本细分为15个节拍（Beat Sheet），适用于电影和剧集单集：

1. **开场画面**（Opening Image）：建立基调
2. **主题陈述**（Theme Stated）：暗示故事主题
3. **建制**（Set-Up）：介绍角色和他们的世界
4. **催化剂**（Catalyst）：改变一切的事件
5. **辩论**（Debate）：主角犹豫是否行动
6. **进入第二幕**（Break into Two）：主角决定行动
7. **B故事**（B Story）：副线（通常是爱情线）
8. **娱乐游戏**（Fun and Games）：承诺的场景
9. **中点**（Midpoint）：虚假胜利/失败
10. **坏人逼近**（Bad Guys Close In）
11. **失去一切**（All Is Lost）
12. **灵魂黑夜**（Dark Night of the Soul）
13. **进入第三幕**（Break into Three）
14. **结局**（Finale）
15. **终场画面**（Final Image）：与开场呼应

### 英雄之旅在权谋剧中的应用
> **来源**: Joseph Campbell《The Hero with a Thousand Faces》(1949)
> Christopher Vogler《The Writer's Journey》(1992)

英雄之旅的12个阶段可以映射到权谋剧中：

| 阶段 | 权谋剧对应 | 藏海传示例 |
|------|-----------|-----------|
| 平凡世界 | 灭门前的平静 | 蒯家日常生活 |
| 冒险召唤 | 发现癸玺/灭门 | 蒯铎发现癸玺 |
| 拒绝召唤 | 试图逃跑 | 蒯铎急返京城避祸 |
| 遇到导师 | 师父高明 | 高明救走稚奴 |
| 跨越门槛 | 改名藏海回京 | "从今天起你叫藏海" |
| 考验/盟友/敌人 | 侯府中周旋 | 杨真为敌，庄之行为突破口 |
| 接近洞穴 | 发现癸玺秘密 | 发现地下密室 |
| 严峻考验 | 身份暴露危机 | 杨真密谋除掉藏海 |
| 获得奖赏 | 获得权力/信息 | 成为内阁重臣 |
| 回去的路 | 真相大白 | 阴谋全貌浮出 |
| 复活 | 最终对决 | 朝堂大对决 |
| 带着灵药返回 | 守护苍生 | 从复仇者到守护者 |

### 三十六计在权谋剧中的应用
> **来源**: 《三十六计》——中国古代兵法策略集

三十六计是权谋剧创作中最常用的策略库，分为六套：

1. **胜战计**: 瞒天过海、围魏救赵、借刀杀人、以逸待劳、趁火打劫、声东击西
2. **敌战计**: 无中生有、暗度陈仓、隔岸观火、笑里藏刀、李代桃僵、顺手牵羊
3. **攻战计**: 打草惊蛇、借尸还魂、调虎离山、欲擒故纵、抛砖引玉、擒贼擒王
4. **混战计**: 釜底抽薪、浑水摸鱼、金蝉脱壳、关门捉贼、远交近攻、假道伐虢
5. **并战计**: 偷梁换柱、指桑骂槐、假痴不癫、上屋抽梯、树上开花、反客为主
6. **败战计**: 美人计、空城计、反间计、苦肉计、连环计、走为上

## 📝 电视剧剧本格式规范

> **来源**: 知乎回答《剧本到底长什么样子?》
> https://www.zhihu.com/question/24525105

### 电视剧剧本格式
电视剧剧本相比电影剧本，台词更多。电视剧剧本在场号前会加上集数数字，如"1-1"表示第一集第一场。

### 标准格式
```
场号-场景号 日/夜 内/外 场景名

（场景描述）

角色名
  （动作/情绪指示）
  台词内容
```

### 示例（参考《潜伏》剧本格式）
```
1-1 日 内 朝堂大殿

藏海独自站在大殿中央，面对龙椅。

藏海
  （低沉）
  陛下，臣有本奏。
```

## 📐 历史权谋分镜设计规则

### 朝堂权谋戏的镜头语言
| 场景 | 分镜方案 | 目的 |
|------|---------|------|
| 朝会大典 | 广角俯视全景(5s) → 慢速横摇扫过群臣(8s) → 皇帝端坐特写(3s) | 展现权力格局 |
| 密室谋划 | 中景双人(4s) → 秘密耳语的特写(2s) → 门外人影一闪(3s) | 营造阴谋氛围 |
| 权力对峙 | 正反打先高速切换(每2s一切) → 逐渐放慢(每4s一切) → 最后锁定面部 | 从激烈到沉着 |
| 阴谋暴露 | 找到证据的手部特写(3s) → 抬头震惊的面部(3s) → 环视四周的快速环绕(5s) | 信息→反应→环境 |
| 策马传信 | 马蹄飞奔的低角度跟拍(4s) → 信使紧张面部的特写(2s) → 抵达目的地的急停(3s) | 紧迫感传递 |

### 古装剧的光影语言
| 人物意图 | 布光方案 | 效果 |
|---------|---------|------|
| 帝王威严 | 高位正面硬光 + 龙椅背景深色 | 不可直视的压迫感 |
| 谋士诡计 | 侧光半明半暗 + 眼神藏在阴影中 | 捉摸不透的神秘 |
| 忠臣直谏 | 正面柔和光 + 面部全部清晰可见 | 坦荡光明 |
| 阴谋关头 | 烛光单一光源 + 影子在墙上放大 | 密室中的紧张 |
| 战场运筹帷幄 | 地图上的斜光 + 手指划过地图的剪影 | 掌控全局 |

### 历史权谋三要素镜头
1. **看（观察）**: 从高处/暗处/屏风后观察他人，镜头带遮挡物，偷窥感
2. **思（思考）**: 面部特写，眼神微动但面部不动，内心盘算可视化
3. **行（行动）**: 行动干净利落，镜头快切，不拖泥带水
