## 📐 核心原则（与动画共用）

### Director's Formula
```
[Subject 主体] + [Action 动作] + [Scene 场景] + [Camera 运镜] + [Style 风格] + [Constraints 约束]
```
**真人视频类型**: Drama/Narrative → 60-100字，以Scene引导

### One-Move Rule（每镜头一个运镜）
❌ 堆叠多个运镜（dolly push + pan + tilt）→ 导致抖动和生成失败
✅ 每镜头只选一个：`locked-off / dolly / pan / tilt / orbit / handheld / crane / tracking`

### Camera Contract（运镜合约）
```
Framing: [wide / medium / close-up / tight close-up]
Move: [locked-off / dolly / pan / tilt / orbit / handheld / crane / tracking] (只选一个)
Speed: [slow / moderate / fast]
Angle: [eye level / low angle / high angle / dutch angle]
```

### 灯光设计规则
- 必须指定命名光源，不用模糊词
- ❌ dramatic lighting → ✅ `candle practical, warm gold, deep shadow beyond 1m`
- ❌ beautiful lighting → ✅ `single overhead practical as hard key, 5600K, deep shadow fill`

### 物理语言
- ❌ "tense atmosphere" → ✅ `breath misting in cold air at each exhale`
- ❌ "epic battle" → ✅ `dust erupts violently, sweat flies off in slow motion`
